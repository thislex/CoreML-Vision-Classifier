# CoreML Vision Classifier

On-device image classification using CoreML and Vision framework.

## Requirements
- iOS 16+
- Xcode 15+

## Setup
- Download MobileNetV2.mlmodel from Apple's model gallery and add to project before building
  https://developer.apple.com/machine-learning/models/

## Stack
- SwiftUI
- Vision framework
- CoreML
- Swift Concurrency

## Architecture 
MVVM - Models, ViewModels, and Views are separated into distinct folders.
